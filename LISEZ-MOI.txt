COMMENT INSTALLER (PowerShell) :
---------------------------------
1. Dézippe ce fichier n'importe où (ex: Downloads\forge-patch)
2. Depuis la racine de ton repo NOXEL FORGE, colle :

   Copy-Item "C:\chemin\vers\forge-patch\*" -Destination "." -Recurse -Force

3. Vérifie avec git status que seulement ces 4 fichiers ont changé.
4. npm run build (dans backend/) pour confirmer que ça compile.

NOUVEL ENDPOINT :
------------------
GET /api/forge/matchmaking/suggestions?limit=20  (auth requise)

Retourne les meilleurs partenaires potentiels, triés par score de
matching (niche + proximité Trust Score + activité + fraîcheur).

Le frontend (ForgePartners.tsx) affiche maintenant une section
"Recommended for you" en haut de la page /forge/partners, avant
l'annuaire filtré existant.


POUR TESTER :
-------------
Il te faut au moins 2-3 membres Forge actifs (badge_installe = true,
statut = 'actif') dans Supabase pour voir des résultats — sinon la
section reste vide silencieusement (comportement normal, pas un bug).
